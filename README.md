# Add Shadow Effect

```
import {shadowizard} from 'shadowizard';
shadowizard({
      shadow_type: 'soft',
      padding: true,
      border_radius: 'round'
    });   
```

# options

* *shadow_type* - hard | soft ( Defaults -> soft)
* *padding* - boolean ( Defaults -> false)
* *border_radius* - round | circle ( no default value)
